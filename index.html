    ABOUT PROJECT
    In this project we have a QnA chatbot which answers questions related to heart failure and heart problems after COVID-19. This chatbot firstly, greet the user and than ask queries and give answers to questions accordingly. For this project i've used Azure QnA maker which is an Azure Coginitive Service, Azure Bot Service and Azure Static Web App.
